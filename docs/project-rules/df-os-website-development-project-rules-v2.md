Df-OS Website Development Project Rules v2

Antigravity 2.0 Execution Guidelines

1. Project Objective

Build an enterprise-grade, premium, ultra-fast, secure, SEO-ready, GEO-ready, AEO-ready, and scalable website for Df-OS.

The website must position Df-OS as a category-defining AI-ready Digital Factory Operating System for modern manufacturing.

The website should not feel like a generic SaaS website, IT services website, dashboard template, or industrial brochure. It must feel like a premium enterprise manufacturing technology platform built for CXOs, plant heads, digital transformation leaders, manufacturing heads, quality leaders, maintenance leaders, operations teams, and enterprise decision-makers.

The website must support five strategic goals:

Create market awareness for the Digital Factory Operating System category.

Position Df-OS as a category creator and leader.

Explain the Df-OS product ecosystem clearly: Df-OS, X-Konnect, Hectos, and Vish AI.

Convert visitors into qualified demo and sales conversations.

Build a robust SEO/GEO/AEO foundation for long-term organic visibility, AI-search discoverability, blog scalability, and thought leadership-led demand generation.

The website is not only a brand website. It must become Df-OS’s long-term organic growth engine.

Df-OS is an AI-ready Digital Factory Operating System that helps manufacturers digitize, connect, automate, and optimize factory operations across people, machines, materials, methods, systems, and decisions.
